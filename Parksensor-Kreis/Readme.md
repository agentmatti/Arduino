# Parksensor

Der Parksensor lässt das LED Band entsprechend dem Abstand vom Sensor leuchten. Die Farbe ändert sich dabei je nach Abstand.

## Bauteile
Du benötigst
- NodeMCU Board
- Widerstand
- WS2812 LED Band mit min 10 LED
- Ultraschallsensor HC-SR-04
